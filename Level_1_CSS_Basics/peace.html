1.Internal Styling

- We do Internal Styling by using style tag in the head.
- Here the styling is implemented on All Occurence of tag defined inside style tag.




2.Inline Styling

-We can apply Inline Styling directly to HTML elements using style Attriute.
- In Inline styling the FIRST PRIORITY is given to INLINE STYLING then to Internal styling.



3. External Styling

Separate CSS File : Stores styles in separate .css file, linked to HTML.
Reusable : Enables style reuse across multiple webpages.
Link in HTML : Use the < link > tag within the < head >section to link the css file.
Relative or Absolute path : The href attribute can contain a relative or absolute path to the CSS file.

8. Universal Selector

THIS TAG HAS LEAST PRIORITY. (Universal selector has limited use, only for those whose property is not specifically defined)
Generally used ot set margin and Padding to zero
Matches All : Targets and styles all elements on a webpage.
Syntax : Utilized as an asterisk(*).
Resets Styles : Commonly used to reset margins and paddings globally.
Broad Styling : Useful for setting Universal attributes like font color.
Usage Caution : Can cause style conflicts due to it's wide-reaching effects.

8. Selectors (Id selectors)

Unique Identifier : targets a specific element with a unique ID attribute.
Syntax : Each ID should be used once per page for uniqueness.
Specific Targeting : Ideal for styling individual, distinct elements.
